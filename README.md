# 🌐 CodeSurfer Pro (Huker AI Edition) v4.1

Welcome to **CodeSurfer Pro**, a high-powered, cyberpunk-themed Chrome extension designed to dynamically inject, execute, and learn Javascript on any webpage using multiple AI providers. This Open-Source edition comes with a **Zero-Config Self-Learning Brain (Huker AI)**, a **Secure Security Vault**, and full support for executing DOM manipulations instantly.

---

## 🔥 Key Features

- **The Hacker Interface:** Matrix-style, neon-green UI inspired by retro cyberpunk terminals.
- **Multiple AI Providers:** Connect your own keys for Groq, OpenRouter, Gemini, Claude, and DeepSeek.
- **Huker AI (Self-Learning RAG Brain):** A unique, semantic-hashing AI engine that *learns* the Javascript code it writes. If you ask for the same action later, it instantly recalls the memory from the DB instead of wasting API tokens.
- **3-Tier Storage Engine:** Configure the AI's Brain to run entirely isolated Locally (Zero-Config), on the Cloud (Firebase), or in Hybrid Sync mode.
- **Master Password Vault:** Your API keys are strictly AES-encrypted under a master password inside Chrome's local storage. Zero hardcoded keys.
- **2-Piston Execution Engine:** Generate custom payloads and inject them directly into the current browser context.

## 🛠️ Installation Guide (Load Unpacked)

Since this is an open-source tool heavily relying on runtime code injection, it is loaded manually into Chrome.

1. Download or clone this repository to your local machine:
   ```bash
   git clone https://github.com/your-username/huker-sufer-pro.git
   ```
2. Open Google Chrome and navigate to `chrome://extensions/`.
3. Enable **Developer mode** using the toggle switch in the top right corner.
4. Click on the **Load unpacked** button.
5. Select the `github_opensource` folder from the cloned repository.
6. Pin the extension to your toolbar for fast access.

## 🔐 Setup & "Bring Your Own Key" (BYOK)

For security, **this extension contains ZERO API keys**. You must provide your own API keys to animate the AI. 

### Step 1: Open the Security Vault
1. Click the Extension icon in Chrome.
2. In the top header next to the title, click the **[⚙️]** button to open the Settings Vault.
3. On first run, create a **Master Password**. This will be used to encrypt your keys.

### Step 2: Add Provider Keys
Inside the Vault, you can enter any API Keys you possess. We highly recommend generating a free **Groq API Key** as it acts as the primary "Semantic Hasher" for the Huker AI engine.
*   **Groq (Recommended Core):** [console.groq.com/keys](https://console.groq.com/keys)
*   **OpenRouter:** [openrouter.ai/keys](https://openrouter.ai/keys)
*   **Gemini:** [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)

Once you enter a key and hit `[> ENCRYPT & SAVE CONFIG_]`, the respective AI Provider will become "Green" (Active) in the main Extension terminal.

## 🧠 The Huker AI Brain (Storage Modes)

The highlight of Version 4 is **Huker AI**, an autonomous RAG (Retrieval-Augmented Generation) system. When you execute generated code, Huker AI categorizes the *Intent* of your prompt and saves the optimized Javascript. 

You can configure where this memory lives in the Vault:

1. **🟡 Mode 1: Local Offline (Default)**
   *   Requires zero configuration. Instantly saves and recalls code from `chrome.storage.local`. High-speed, high-privacy, but data does not sync across devices.
2. **🔵 Mode 2: Firebase Cloud**
   *   Create a Realtime Database on [Firebase](https://firebase.google.com/).
   *   Set the rules to allow public or authenticated R/W.
   *   Copy your `https://your-project.firebaseio.com/memories.json` URL into the Vault.
   *   Persistent sync across any laptop you use the extension on.
3. **🟢 Mode 3: Hybrid Pro**
   *   Combines the instant recall speed of Local Storage with the permanent background backup sync of Firebase.

## ⚠️ Security & Privacy Warning

*   **Never Push Keys:** Do not hardcode your API keys back into `config.js` or `huker-ai.js` if you plan to fork or commit. Always use the secure Vault UI.
*   **Execution Risk:** The extension executes dynamically generated Javascript in your active browser tab. Do not run payloads generated by the AI on sensitive websites (like banking) unless you understand the generated code.
